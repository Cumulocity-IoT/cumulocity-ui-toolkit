# Cumulocity Advanced Radial Gauge Widget Plugin

This Advanced Radial Gauge widget plugin is used to view the latest realtime measurement on a radial gauge by defining the measurement properties in config page.


| Field              | Description                                                            |
| ------------------ | ---------------------------------------------------------------------- |
| Fragment           | Fragment of the measurement.                                           |
| Series             | Series of the measurement                                              |
| Label              | The label of the measurement.                                          |
| Unit               | Measurement unit                                                       |
| Range Min          | Minimum gauge range.                                                   |
| Range Max          | Maximum gauge range.                                                   |
| Orange Range Start | The start range at which orange color starts.                          |
| Red Range Start    | The start range at which red color starts.                             |
