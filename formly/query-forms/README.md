# Query Forms

Gives the user the opportunity to create [query filters](https://cumulocity.com/api/core/#tag/Query-language) using an easy to use UI.

## Create an Alarm query filter

<img src="assets/ps-alarm-query-form-example.png" width="800" />

```typescript
<ps-alarm-query-form
    [filter]="filter"
></ps-alarm-query-form>
```
